Hello!

This website was put together by Thomas J. Fox for an HONR 216 Encountering Contemporary Europe class at Loyola University Chicago.

The point of this project is to inform the user of the basic history and workings behind the Enigma Cipher Machine that was in use during the Second World War by Nazi Germany. The encryption machine itself is a portable rotating shift cipher that makes it incredibly difficult to crack. It was a large advantage for the German war effort over the Allied Forces during WWII. At the time of its use, it was thought to be impossible to outright crack the code. There are 158,962,555,217,826,360,000 (158 quintillion 962 quadrillion 555 trillion 217 billion 826 million 360 thousand) different possiblities for a letter to be encrypted to using this algorithm.

To learn more, and then to encrypt your own message using the same algorithm used by the Germans in the 1940s, check out the history section and then the enigma machine.
